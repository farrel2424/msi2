Environment Setup

üìã Required Credentials

You need two API keys to use the Motorsights EPC automation:

Option 1: Environment Variables (Recommended)

Create a .env file in the project directory:

SUMOPOD_API_KEY=sk-your-sumopod-key-here EPC_BEARER_TOKEN=your-epc-token-here

Option 2: Web Interface

Enter keys directly in the upload form (less secure but convenient for testing)

Configuration Parameters

Sumopod API Key

Variable: SUMOPOD_API_KEY

Your API key for Sumopod AI Gateway. Get this from your mentor.

Format: sk-xxxx...

EPC Bearer Token

Variable: EPC_BEARER_TOKEN

Authentication token for Motorsights EPC API. Get this from your mentor.

Master Category ID (Required)

Variable: DEFAULT_MASTER_CATEGORY_ID

UUID of the default master category for all uploaded items.

Starting the Web UI

Run the following command:

python epc_web_ui.py

Open your browser to:

http://localhost:5000
